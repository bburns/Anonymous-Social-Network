Help and Support

Well, you're here... so something must have gone wrong. Right now we are still building the site, and there may be some problems here and there, so please bear with us.

To test out the user functionality of the site, login with your userid and password as given in your XML instance.

To test out the administrator functionality of the site, login with username admin000 and password admin. You will also be a student user.

Note that we moved some of the user functionality to admins - we thought only admins should have the ability to edit and delete items. Users can still edit and delete their ratings and comments for items though.

Note that in order to test out the import functionality, we do a semi-fuzzy search on existing items, in order to get the aggregate ratings and grades. This is a bit artificial, as if all the data were generated from the site and then exported to XML, we could do exact matching. So in importing games, for instance, if a game with the same title already exists, it will use that, and ignore the operating system value. For classes, it matches on the course number exactly, then checks if the instructor has a partial match, ignoring the course name.

We store Semester and Unique# in the rating object, so that the average rating and grade will be taken for a given course and instructor combination.

For more information, see our Google Code site, with issue tracker, here. If you find a bug, please submit a bug report!
